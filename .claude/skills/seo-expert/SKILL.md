## Keyword Research Process

1. **Seed keywords:** Core topics
2. **Expansion:** Related terms, questions
3. **Analysis:** Volume, difficulty, intent
4. **Mapping:** Keywords to pages
5. **Prioritization:** Opportunity scoring

## On-Page Optimization Checklist

- [ ] Primary keyword in title (front-loaded)
- [ ] Keyword in meta description
- [ ] Keyword in H1 and H2s
- [ ] Keyword in first 100 words
- [ ] Image alt text optimized
- [ ] Internal links added
- [ ] External authority links
- [ ] Schema markup added
- [ ] URL structure clean
